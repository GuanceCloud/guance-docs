# 获取当前工作空间信息

---

<br />**get /api/v1/workspace/get**

## 概述
获取当前调用者所属工作空间信息




## 参数补充说明





## 请求例子
```shell
curl 'https://openapi.guance.com/api/v1/workspace/get' \
-H 'DF-API-KEY: <DF-API-KEY>' \
--compressed \
--insecure
```
