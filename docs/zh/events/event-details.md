# 事件详情
---

## 简介

在未恢复事件查看器或者事件查看器中，点击任意事件，即可查看事件详情，包括基础属性、告警通知、状态&趋势、历史记录、关联事件和关联 SLO 等。

## 事件详情页

在事件详情页，您可以：

- 点击右上角的**跳转到监控器**，可查看和调整 [监控器配置](../monitoring/monitor/index.md)；  
- 点击右上角的**导出**按钮，支持选择**导出 JSON 文件**与**导出 PDF 文件**，从而获取当前事件所对应的所有关键数据。

### 基础属性 {#attribute}

显示查看事件的检测维度、事件内容以及扩展属性。

![](img/5.event_8.png)

观测云支持关联查询所有检测维度。在**检测维度**，点击**关联查询**，即可查看相关检测内容。


![](img/event-1.png)


### 告警通知

显示通知对象类型、通知对象名称、通知是否发送成功等信息。点击展开显示该告警通知对象的详细信息，支持 hover 复制。

???+ attention

    在静默期间，告警通知不会重复发送至相关对象。

![](img/5.event_10.png)

### 状态&趋势

显示查看事件的状态分布趋势、DQL 函数和窗口函数折线图。

![](img/5.event_9.png)

- 状态分布：展示选定时间范围内（默认展示最近6小时）的事件状态 (紧急、重要、警告、无数据)；  
- DQL 查询语句：基于异常检测规则的自定义查询语句返回的实时指标数据，默认展示最近6小时的实时指标数据；  
- Window 函数：基于异常检测规则，以选定的时间范围为窗口（记录集合），以检测频率为偏移，重新对每条记录执行统计计算，返回用于触发告警的实时异常检测指标数据。默认展示最近 6 小时的实时异常检测指标数据。  

![](img/image.png)

???+ attention "在事件详情中，观测云支持选择时间范围查看事件数据"

    - 当您选择的时间范围小于 ( < = ) 6小时，**状态分布**、**DQL 函数**、**Window 函数**将展示当前时间范围的数据与指标趋势；  
    - 当您选择的时间范围大于 ( > ) 6小时，**状态分布**与 **DQL 函数**将展示当前时间范围的数据，且出现一个可调节的区间滑块（显示范围最小支持 15 分钟，最大支持 6 小时）。通过移动区间滑块，可查看与之时间范围对应的 **Window 函数**。 

### 历史记录

显示查看检测对象主机、异常/恢复时间和持续时长。

![](img/5.event_11.png)

### 关联信息

显示查看触发当前事件的相关信息，如查看触发事件的相关日志。此<u>关联信息</u>仅支持 4 种监控器产生的事件：<u>日志检测、安全巡检异常检测、进程异常检测以及可用性数据检测</u>。

???+ attention

    若日志检测包含多个表达式查询，关联信息支持多个表达式查询的 tab 切换，若有两个表达式查询 A 和 B，则在关系信息包含 A 和 B 两个 tab 可切换查看。

![](img/3.event_13.png)

### 关联事件

支持通过筛选字段和所选取的时间组件信息，查看关联事件。

![](img/5.event_12.png)

### 关联仪表板

若在监控器配置了关联仪表板，则可以查看关联的仪表板。

![](img/5.event_13.png)



### 关联 SLO

若在监控配置了 SLO ，则可以查看关联 SLO ，包括 SLO 名称、达标率、剩余额度、目标等信息。

![](img/5.event_14.png)

## 聚合事件详情页

在事件查看器中，选择分析维度以后（如监控器 ID），支持查看基于该维度的聚合事件。点击聚合事件，即可查看对应的聚合事件详情，包括基础属性、状态&趋势、告警通知、关联事件和关联仪表板等。详情介绍见上文。

![](img/5.event_15.png)
