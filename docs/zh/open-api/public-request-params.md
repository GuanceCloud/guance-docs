# 公共请求参数

---

本文介绍 Open API 的公共请求参数。 

## 公共请求头(Header)

| 参数名        | 类型      | 说明                                                                |
|:-----------|:--------|:------------------------------------------------------------------|
| Content-Type | string  | application/json                                                  |
| DF-API-KEY | string  | 调用者标识, 获取方式见[API Key 管理](../management/api-key/open-api.md)|
