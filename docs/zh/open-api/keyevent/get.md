# 获取一个事件的内容

---

<br />**get /api/v1/events/\{doc_id\}/get**

## 概述
根据`__docid`获取指定事件内容




## 路由参数

| 参数名        | 类型     | 必选   | 说明              |
|:-----------|:-------|:-----|:----------------|
| doc_id | string | Y | 事件ID<br> |


## 参数补充说明





## 请求例子
```shell
curl 'https://openapi.guance.com/api/v1/events/E_c7jodqs24loo3v53qc30/get' \
-H 'DF-API-KEY: <DF-API-KEY>' \
--compressed 
```
