# 删除一个拨测任务

---

<br />**post /api/v1/dialing_task/delete**

## 概述




## Body 请求参数

| 参数名        | 类型     | 必选   | 说明              |
|:-----------|:-------|:-----|:----------------|
| taskUUIDs | array | Y | 云拨测任务的UUID列表<br>允许为空: False <br> |

## 参数补充说明





## 请求例子
```shell
curl 'https://openapi.guance.com/api/v1/dialing_task/delete' \
-H 'DF-API-KEY: <DF-API-KEY>' \
-H 'Content-Type: application/json;charset=UTF-8' \
--data-raw '{"taskUUIDs":["dial_408e8fafe88a49a5ba1cfd73910edfdb"]}' \
--compressed 
```

{
    "code": 200,
    "content": {},
    "errorCode": "",
    "message": "",

}
