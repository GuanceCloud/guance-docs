# 获取标签信息

---

<br />**GET /api/v1/tag/\{tag_uuid\}/get**

## 概述
获取标签详情




## 路由参数

| 参数名        | 类型     | 必选   | 说明              |
|:-----------|:-------|:-----|:----------------|
| tag_uuid | string | Y | 标签UUID<br> |


## 参数补充说明

数据说明

- 响应参数说明

| 参数名           | type | 说明                                                 |
| ---------------- | ---- | ---------------------------------------------------- |
| name       | string | 标签名称 |
| description             | string | 字段描述信息                                                 |
| color    | string | 前后端约定的颜色类型, default,style_key1,style_key2 ~~                |
