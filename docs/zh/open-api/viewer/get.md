# 获取一个查看器

---

<br />**get /api/v1/viewer/\{viewer_uuid\}/get**

## 概述
获取查看器详情




## 路由参数

| 参数名        | 类型     | 必选   | 说明              |
|:-----------|:-------|:-----|:----------------|
| viewer_uuid | string | Y | 查看器UUID<br> |


## 参数补充说明





## 请求例子
```shell
curl 'https://openapi.guance.com/api/v1/viewer/dsbd_e493a3c17d3c456bb1febfcbbe4148d2/get' \
-H 'DF-API-KEY: <DF-API-KEY>' \
--compressed \
--insecure
```
