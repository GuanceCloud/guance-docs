# 删除一个/多个黑名单

---

<br />**GET /api/v1/blacklist/delete**

## 概述
删除一个/多个黑名单




## Query 请求参数

| 参数名        | 类型     | 必选   | 说明              |
|:-----------|:-------|:-----|:----------------|
| blacklistUUIDs | commaArray | Y | 黑名单过滤规则uuid们<br>允许为空: False <br> |

## 参数补充说明





## 请求例子
```shell
curl 'https://openapi.guance.com/api/v1/blacklist/delete?blacklistUUIDs=blist_d221f03ac39d468d8d7fb262b5792607' \
-H 'DF-API-KEY: <DF-API-KEY>' \
--compressed 
```
