# 资源、系统要求


## 阿里云资源清单 {#list}

???+ warning "注意"

    1. “**最低配置**” 只适合 POC 场景部署，只作功能验证，不适合作为生产环境使用。
    1. “**推荐配置**” 适合 InfluxDB 少于15 万时间线，Elasticsearch 少于 70 亿文档数（日志、链路、用户访问监测、事件等文档数总和）的数据量场景使用。
    1. 作为生产部署以实际接入数据量做评估，接入的数据量越多，InfluxDB、Elasticsearch 的存储与规格配置相应也需要越高。
