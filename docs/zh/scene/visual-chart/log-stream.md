# 日志流图 
---

以时序表格图的形式展示日志数据时，您可以自定义日志的显示范围和列配置。此外，系统支持对硬件、软件、系统信息以及事件等日志数据进行搜索和筛选，并以清晰的表格形式展示结果。

![](../img/log.png)

## 图表查询

1. 查询：支持数据源固定为**日志**的查询，支持关键字搜索；  
2. 筛选：通过输入筛选条件过滤查询的日志数据；  
3. 显示列：选择要展示的显示列。`message` 内容默认存在，可重命名内容。
