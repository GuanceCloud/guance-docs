# 视频
---

## 简介

观测云支持添加视频至视图中，可用于教程、说明等等。填入视频地址即可，上方可实时预览视频详情。

## 应用场景

观测云的视频图表用于展示视频。可用于以下场景：

- 展示不同场景的教学视频；
- 针对需要特殊说明提示的视频说明。

## 图表样式
| 选项 | 说明 |
| --- | --- |
| 图表标题 | 为图表设置标题名称，设置完成后，在图表的左上方显示，支持隐藏。 |
| 音量 | 设置视频音量大小。 |
| 自动播放 | 设置视频是否自动播放。 |
| 循环播放 | 设置视频是否循环播放。 |
