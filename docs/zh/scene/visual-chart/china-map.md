# 中国地图
---

用于显示某个指标数据在不同省份的数据分布情况。

## 应用场景

1. 查看包含地理位置信息的数据；
2. 查看分散分布的数据，例如：人口密度；
3. 通过颜色深浅比较数据大小。

## 图表查询

**前提条件**：指标查询的数据集中必须包含表示 `省份` 的标签，并且查询条件中必须包含 `省份` 的分组，否则图表无法添加成功。

图表查询支持**简单查询**、**表达式查询**、**PromQL 查询**和**数据源查询**；您可以添加多个查询，但 by（分组）的 Tag 必须一致。

> 更多详细说明，可参考 [图表查询](chart-query.md)。

图表查询通过省份匹配地区，通过指标值匹配颜色：

| 选项 | 说明 |
| --- | --- |
| 省份标签 | 选择表示省份的标签，该标签必须为查询中已分组的 tag 标签。<br />:warning: 省份标签的标签名不一定非要是`省份`，只要值是省级行政区即可。 |
| 指标 | 当添加了多个指标查询时，可设置主要展示的指标。“主展示指标”决定色块渐变色的指标。 |
