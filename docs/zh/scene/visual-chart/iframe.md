# IFrame
---

用于在仪表板中嵌入外部页面，支持 https 和 http 链接。

![](../img/ifram.png)

## 图表设置

添加 IFrame URL：在图表中内嵌另一个网站。（使用 HTTPS URL 或者将您的浏览器配置为允许不安全的内容）
