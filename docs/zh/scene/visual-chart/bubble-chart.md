# 气泡图
---


气泡图可用于展示三个变量之间的关系，与散点图类似，分为横轴和纵轴，并加入表示大小的变量进行对比。表示因变量随自变量而变化的大致趋势，由此趋势可以选择合适的函数进行经验分布的拟合，进而找到变量之间的函数关系。可用来观察数据的分布和聚合情况。

![](../img/bubble.png)

## 图表查询

分别定义 X 轴与 Y 轴的筛选条件。可切换为表达式查询、PromQL 查询和其他数据源查询。
