# 快照历史版本

<font size=2>**注意**：该入口仅具备[场景配置管理权限](../../management/role-list.md)的成员可见。</font>

<<< custom_key.brand_name >>>会展示当前仪表板最近 3 个月的保存记录，进入页面后可查看每个版本的更新时间、更新人和更新记录。

## 历史版本记录

点击进入历史版本，右侧列出所有版本，默认选中当前版本；左侧图表展示选中版本下的图表内容。可在页面右侧查看当前仪表板不同版本的 JSON 变更记录：

- 红色标识：被删除的内容；
- 绿色标识：新增的内容。

针对版本记录，您可进行以下操作：

- 点击版本右侧的 :material-dots-vertical: 标识，选择还原此版本或克隆仪表板；
- 点击 :material-text-search:，查看该版本的变更详情。

## 视图详情

历史版本模式下，您可以：

- 下拉选择页面上方的视图变量；
- 在时间控件选择时间范围；
- 预览、下载、查看图表。

???+ warning "注意"

    - 若图表显示为空，可能是因为当前版本中没有可用数据；  
    - 图表无法在历史版本模式下修改。
