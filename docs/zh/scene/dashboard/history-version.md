# 快照历史版本

???+ warning "注意"

    该入口仅具备[场景配置管理权限](../../management/role-list.md)的成员可见。

展示当前仪表板最近 3 个月的保存记录，进入页面后可查看每个版本的更新时间、更新人和更新记录。

## 历史版本记录

点击进入历史版本，右侧列出所有版本，默认选中当前版本，左侧显示对应图表。右侧显示不同版本的 JSON 变更记录：红色表示删除内容，绿色表示新增内容。

针对版本记录，您可进行以下操作：

- 点击版本右侧的 :material-dots-vertical: 标识，选择还原此版本或克隆仪表板；
- 点击 :material-text-search:，查看该版本的变更详情。

## 视图详情

历史版本模式下，您可以：

- 下拉选择页面上方的视图变量；
- 在时间控件选择时间范围；
- 预览、下载或查看图表。

???+ warning "注意"

    - 若图表显示为空，可能是因为当前版本中没有可用数据；  
    - 图表无法在历史版本模式下修改。
