
# 列表管理

针对当前工作空间内的所有仪表板，您可通过以下操作进行管理。

## 批量操作

即选定仪表板进行批量删除或导出。

## 修改/导出/删除

点击仪表板 :material-dots-vertical: 按钮，即可修改仪表板名称、导出仪表板到 JSON 文件和删除仪表板。

## 设置主页仪表板

即将仪表板设置为工作空间级别的【主页仪表板】，设置完成后，后续成员访问该空间时默认显示主页仪表板。

**注意**：仅支持对 “公开” 可见的仪表板设置【主页仪表板】，且设置完成后，不支持修改可见范围。

![](../img/dash_01.gif)


## 过滤/筛选

在仪表板左侧，您可以通过不同分类来快速过滤查找对应的仪表板。

同时您还可以通过为仪表板设置标签来进行快速筛选仪表板。

![](../img/dashboard006.png)

| 分类      | 说明         |
| ----------- | ------------------- |
| 我的收藏      | 当前工作空间登录用户收藏的仪表板，点击仪表板 :octicons-star-24: 收藏图标即可。         |
| 导入项目      | 当前工作空间所有通过**导入自定义模板**创建的仪表板。         |
| 我的创建      | 当前工作空间登录用户创建的所有仪表板，包括导入的仪表板。         |
| 仅自己可见      | 仅仪表板创建人可查看的**非公开仪表板**，其他成员不具备查看权限。         |
| 经常浏览      | 当前工作空间登录用户一周内浏览次数大于 5 次的仪表板。         |


## 轮播列表 {#player}

如果您需要同时处理多个关联的业务仪表板，可以使用轮播的功能。配置关联的仪表板后，方便您的一站式业务分享。

![](../img/snapshot-14.png)

1. 进入**轮播列表**，点击**新建轮播**；
2. 输入轮播名称；
3. 选择仪表板；
4. 设置刷新频率；
5. 点击确定。

<img src="../../img/1.dashboard_player_2.png" width="60%" >

配置完成后，即可在**轮播列表**查看已经设置的仪表板。

针对轮播列表，您可采取以下操作：

1. 点击播放按钮，已配置的仪表板即按照设置的频率进行播放；
2. 点击编辑按钮，即可修改该轮播配置；
3. 点击删除，即可删除该轮播。


## 历史快照 {#history}

在页面右上方**历史快照**处可查看当前工作空间保存的所有仪表板快照。
