# 定时报告 {#report}

观测云支持您将仪表板导出为报表，通过多种渠道，根据每日/每周/每月等不同时间维度定时推送给相关人员。

进入**场景 > 定时报告**：

<img src="../img/report-9.png" width="60%" >

或**场景 > 仪表板 > 定时报告**：

![](img/report-1.png)

在当前页面，您可查看定时报告名称、所关联的仪表板、报告周期、通知对象等信息。

![](img/report-2.png)

## 新建报告

![](img/report-3.png)

1、**基本信息**：

![](img/report-4.png)

| 字段      | 描述               |
| ----------- | ---------------- |
| 仪表板      | 选择您需要发送定时报告的相关仪表板。               |
| 修改视图变量      | 您可以选择自定义修改定时报告中仪表板的视图变量。<br/>**注意**： <br/><li>此处的修改只影响报告，不会影响仪表板原有默认值配置；<br/><li>若仪表板中没有视图变量，则不会向您显示该入口。                 |
| 查询范围     | 即发送报告时仪表板查询的时间范围；默认选中【最近 1 天】，您可以手动输入时间范围。               |

2、**定时计划**：

![](img/report-5.png)

| 字段      | 描述               |
| ----------- | ---------------- |
| 时区      | 时区默认显示 `UTC+08:00`，暂不支持修改。               |
| 报告时间      | 即发送当前定时报告的时间；您需填入正整数，范围：时 0-23；分 0-59。               |
| 报告周期      | 即当前定时报告的发送频率周期；包含【一次性】【按天】【按周】【按月】，您可按需选择。<br/><li>一次性：若勾选，当前报告只在您选中的特定时间发送；<br/><li>按天：您可勾选每天；或您可勾选发送当前定时报告的具体日期，支持多选；<br/><li>按周：您可根据【周】的时间维度选择定时发送；<br/><li>按月：您可勾选每月；或您可勾选发送当前定时报告的具体月、日，支持多选。              |

3、**通知方式**：

目前支持 4 种通知类型：钉钉机器人、企业微信机器人、飞书机器人、邮件组。

![](img/report-6.png)

| 字段      | 描述               |
| ----------- | ---------------- |
| 通知对象      | 即当前定时报告的接收人；您可以前往**监控 > 通知对象管理**创建新的通知对象。               |
| 标题      | 即当前定时报告在邮件中显示的名称。               |
| 内容      | 即当前定时报告在邮件中显示的内容。               |

点击确定后，您可以在定时报告列表中查看新建的报告：

![](img/report-7.png)

4、**分享方式**：

![](img/)

- **公开分享**：选择公开分享的定时报告的短链接，可以直接打开；

- **加密分享**：需设置 4～8 位英文、数字的输入密码。设置完成后，需正确输入密码才能打开定时报告短链接。

## 报告列表

在报告列表，您可进行以下操作：

- Hover 至仪表板名称，显示详细信息，包含仪表板名称及视图变量；可点击跳转至对应仪表板页面；
- Hover 至报告周期和通知对象，显示对应详细信息；
- 在报告右侧的**操作**，您可以查看该条定时报告的创建人，还可以选择禁用、编辑或删除该条定时报告。
