# 场景
---

“场景”功能以图表为核心，帮助您灵活地展示和分析数据。您可以自定义编辑和筛选数据，也可以使用内置的多种数据监控视图模板，快速创建仪表板和查看器，或者从零开始构建个性化的数据洞察场景。
