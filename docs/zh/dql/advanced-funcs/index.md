# 高级函数

## 概述	

高级函数用于观测云在**场景**中选择**时序图**，点击**查询**下的**添加函数**按钮，高级函数列表用于对 DQL 查出数据后进一步函数计算，进行直观的时序展示，例如对时序数据进行异常检测、预测、离群分析等各种时序算法分析。

![](../img/ad-1.png)


您可以使用观测云为您提供的**中心函数**，或者您可以在配置**本地函数**，在时序图中使用。
