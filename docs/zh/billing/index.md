# 付费计划与账单
---

## 版本说明

提供体验版、商业版、私有云部署版三种版本。

- 公有云：体验版、商业版。均采用<font color=coral>**按量付费**</font>的计费方式，核心功能无差异，但后者支持更大规模数据接入及更灵活存储时效。

- 私有云部署版：社区版（体验版）与商业版。后者可选择按量付费、订阅制、许可证制等多种计费方式。


## 付费计划与账单 {#billing}

<<< custom_key.brand_name >>>公有云版本秉承按需购买、按量付费的原则。

在付费计划与账单模块，可查看结算账户余额、计费项使用统计数据，并从账单明细、月账单、使用量、数据转发使用三个方面进行费用分析。

### 结算账户总览 {#account}

查看结算账号名称、现金账户余额、代金券余额、预购卡余额，并对账户进行高消费预警设置，或直接前往[费用中心](../billing-center/index.md)。

#### 设置高消费预警 {#alert}

???+ warning "注意"

    仅 Owner 和 Administrator 可操作。

- 总阈值预警：为工作空间设定总体消费阈值，超阈值时向成员发送邮件预警。

- 单个计费项预警：针对单个计费项设置预警，日账单金额超阈值时发送邮件预警。

<img src="img/billing.png" width="50%" >

1. 填写总预警阈值；
2. 选择[计费项](../billing-method/billing-item.md#item)并输入**预警阈值**；
3. 选择通知成员，无论总阈值还是单一计费项的实际账单金额超出设定的预警阈值时，都会向成员发送预警邮件；
4. 点击**确定**。

???+ warning "注意"

    Owner 角色不受通知对象选择的限制，触发预警时会收到邮件通知。



### 使用统计

查看各计费项**截止当前**和**截止昨日**的统计数据。

> 关于计费项和定价模型，可查看 [计费方案](../billing-method/index.md)。


### 账单明细

以出账模式、使用量、应付金额、欠费金额等维度展示各产品消费数据，可按月份查看费用统计。

支持一键“隐藏 0 元账单”，可导出所选月份或近 12 个月的账单明细为 CSV 文件。

### 月账单

以月为单位展示账单总量统计，维度包括月消费合计、现金消费、云账号消费、代金券消费、预购卡消费、欠费合计。

支持一键“隐藏 0 元账单”，可导出所选月份或近 12 个月的账单明细为 CSV 文件。

### 使用量分析

以折线图图表的形式可视化展示各计费项目使用情况。


### 数据转发使用分析 {#transmit}

查看工作空间所有数据转发规则的转发数量，可按今日、昨日、本周、上周、本月、上月、今年等时间维度查看统计。

???+ warning "注意"

    - 若转发规则保存到<<< custom_key.brand_name >>>的备份日志，则显示对应的数据保存策略，否则展示为 `-`。
    - 仅列出存在且转发数量 > 0 的规则。


## 费用中心

???- warning "工作空间角色差异说明"

    - 拥有者：有**费用中心**按钮；   
    - 管理员：无上述按钮；  
    - 其他成员：无**付费计划与账单**模块查看权限。

<<< custom_key.brand_name >>>为商业版用户提供两套独立运行、统计数据量相关连的账号体系，实现用量计费与费用结算：

- [控制台](https://<<< custom_key.studio_main_site >>>/)账号，统计工作空间数据量接入规模、账单明细等，同步到指定的费用中心账号。
- [费用中心](https://<<< custom_key.boss_domain >>>/)账号，通过 `工作空间 ID` 绑定，实现工作空间级别的统一费用管理，支持多种费用结算方式。
