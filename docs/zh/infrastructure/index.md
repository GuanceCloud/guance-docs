# 基础设施
---

## 概述

观测云支持采集主机，容器，进程，网络和其他自定义对象的数据并上报到工作空间。在**基础设施**页面，您可以通过所属的基础设施分类列表查看所有对象的数据。

例如：查看主机 A 的 CPU、内存等指标数据，可以前往[**主机**](./host.md)；查看容器 B 的 CPU 数据可以前往**[容器](./container.md)**。
