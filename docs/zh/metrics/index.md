# 指标
---


{{{ custom_key.brand_name }}}具有全域数据采集能力，由采集器获取的指标数据会自动上报至工作台，您可以通过**指标**对空间内的指标数据进行统一的分析和管理。

## 应用场景

确保系统的可观测性、可维护性和可运营性对组织来说至关重要，它们是保障业务连续性和服务交付的关键。因此，采用广泛的指标来衡量系统、应用和业务的运行状态及服务效率变得尤为关键。

1. 实现各类技术栈、开源软件、业务上的指标观测需求；
2. 可视化分析指标变动趋势；
3. 统一管理指标集、指标、标签。
