# 云厂商

---

[DataKit](../../datakit/) 是观测云中至关重要的一个数据采集组件，几乎所有观测云中的数据都是来源于 DataKit。

[DataFlux Func](../../dataflux-func/) 是观测云的一个基于 Python 的脚本开发、管理、执行平台。
