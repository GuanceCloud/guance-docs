# Snapshot History Versions

???+ warning "Note"

    This entry is only visible to members with [scenario configuration management permissions](../../management/role-list.md).

This displays the save records of the current dashboard for the past 3 months. After entering the page, you can view the update time, updater, and update records for each version.

## Historical Version Records

Click to enter the historical version, where all versions are listed on the right side. The current version is selected by default, and the corresponding charts are displayed on the left. On the right, the JSON change records of different versions are shown: red indicates deleted content, green indicates added content.

For version records, you can perform the following actions:

- Click the :material-dots-vertical: icon on the right side of the version to restore this version or clone the dashboard;
- Click :material-text-search:, to view the change details of this version.

## View Details

Under historical version mode, you can:

- Use the drop-down menu to select view variables at the top of the page;
- Select a time range using the time widget;
- Preview, download, or view the charts.

???+ warning "Note"

    - If the chart displays as empty, it may be because there is no available data in the current version;  
    - Charts cannot be modified under historical version mode.
