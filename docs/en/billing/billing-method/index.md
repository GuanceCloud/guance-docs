## Overview

When you open the workspace of Commercial Plan and start using it, Guance provides a <font color=coral>pay-as-you-go billing method</font>. This article mainly introduces the detailed information of pay-as-you-go for Guance products.


## Billing Items {#item}

All charging items of Guance are charged separately. For example, the log data you report will incur the cost of log storage, and the application performance link tracking data you report will incur the cost related to application performance trace.

Detailed billing items of Guance and their corresponding statistical scope and billing types are as follows:

<font color=coral>**Note:**</font> You can view the statistics of each billing item, historical billing, usage statistical trends and other information of the previous day in the Guance **Console > Billing**.

### Logs {#logs}

#### :material-numeric-1-circle-outline: Billed by the number of data pieces.

The data generated by the use of functions such as logs, events, security check, and availability dialing (per million records/days).

??? Note

    - If the Custom Multi-Index feature is enabled for logs, the data will be counted based on different indexes to calculate the actual cost according to the corresponding data storage pricing strategy.    
    - Events include events generated by monitoring modules (monitoring, SLO) configuration detection tasks, events reported by intelligent inspections, and events reported by users.     
    - Availability testing data is reported by self-built testing nodes.        
    - The cost of events, security inspections, and availability testing data is calculated based on the data storage pricing strategy of the "Default" index for logs.

## FAQ

:material-numeric-1-circle: Why should we distinguish between RMB settlement and USD settlement?

In response to the different needs of customers domestic and overseas，Guance has launched two sets of settlement center systems, [the domestic version](https://boss.guance.com/) and [overseas version](https://bill.guance.one/). The former is settled in RMB, while the latter is settled in US dollars. In each system, there will be different pricing for different site logins, see above for details.

:material-numeric-2-circle: What should I do if I want to change the currency settlement method, for example, from RMB settlement to USD settlement?

Please contact your account manager directly.
