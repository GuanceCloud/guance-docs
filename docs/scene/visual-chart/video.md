# 视频
---

## 简介

“观测云” 支持添加视频至视图中，可用于教程、说明等等。填入视频地址即可，上方可实时预览视频详情。

## 图表样式
| 选项 | 说明 |
| --- | --- |
| 图表标题 | 为图表设置标题名称，设置完成后，在图表的左上方显示，支持隐藏 |
| 音量 | 设置视频音量大小 |
| 自动播放 | 设置视频是否自动播放 |
| 循环播放 | 设置视频是否循环播放 |
