# 图片
---

## 简介

图片一般用于展示图片，可以在图片中添加图片地址即可显示对应图片，可通过选择填充、适应、拉伸、居中调整图片形状。在链接页面可添加需要跳转的网址。

## 图表样式
| 选项 | 说明 |
| --- | --- |
| 图表标题 | 为图表设置标题名称，设置完成后，在图表的左上方显示，支持隐藏 |
| 方式 | <br />- 填充：图片等比缩放，之后按照图片的最小边来适卡片的最大边以达到填充屏幕效果<br />- 适应：图片等比缩放，只不过图片的最大边放大到卡片最小边时就不再放大，保持图片比例的同时最大化显示图片；<br />- 拉伸：图片不按比例缩放，而是根据卡片大小进行拉伸<br />- 居中：图片大小不变居中显示<br /> |
